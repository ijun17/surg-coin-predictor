## 1. 타켓 코인 선정

`coin_selector.py`

업비트 API로 모든 코인 데이터를 받아와 조건에 맞는 코인을 선별

필터 : 원화 마켓만, 가격이 낮음, 거래량이 적음

## 2. 코인 데이터 수집

`coin_collector.py`

업비트 API로 타겟 코인의 월, 주, 일, 분(240/60) 별 데이터 수집
